步骤二：添加库

至此，我们已经知道了如何使用CMake创建一个基本项目。在这一步，我们将学习如何在创建库以及如何在项目中使用，也将了解到如何选择使用我们的库。

练习一：创建库

要在CMake中添加库，使用 add_library() 命令并指定打包到库里的源文件。

我们用一个或者多个子目录来组织项目，而不是将所有源文件放到一个目录下。这里为我们库创建一个特定的子目录。我们增加一个新得 CMakeLists.txt 文件和一个或多个源文件。 在 CMakeLists.txt 文件头部，使用 add_subdirectory() 命令增加要构建的子目录。

当库构建成功，通过 target_include_directories() 和 target_link_libraries() 命令链接到可执行程序。

练习二：使创建的库为可选

现在设置我们的数学方法库为可选。虽然对于教程来说，确实没必要这样做，但对于大型项目来说，这是经常会遇到的情况。

CMake可以使用 option() 方法设置库为可选。这为用户提供了一个变量，用户可以在配置CMake构建时更改该变量。设置将存储在缓存中，因此用户不需要每次在构建目录内运行CMake时重新设置。

目标

添加不使用 MathFunctions 的构建选项。

解决方案

第一步是在 CMakeLists.txt 文件顶部增加选项。在 cmake-gui 和 cmake 中，option会设置默认值为 ON，用户可以修改变量值。

TODO 7: Click to show/hide answer
TODO 7: CMakeLists.txt
option(USE_MYMATH "Use tutorial provided math implementation" ON)

接下来，可选择性的构建和链接 MathFunctions 库。

首先为我们的项目可选库构建 list()，目前只有 MathFunctions。列表命名为 EXTRA_LIBS。
类似的，对于可选头文件构建列表 list() 命名为 EXTRA_INCLUDES。在列表中，使用 APPEND 追加头文件路径到需要的库中。

然后，使用 if() 命令行判断 USE_MYMATH 的状态。在 if() 代码块内，将练习一的 add_subdirectories 命令作为 list() 命令的可选项。

当 USE_MYMATH 为 ON，将生成列表并添加到项目中。当 USE_MYMATH为 OFF，列表为空。使用次策略，允许用户切换 USE_MYMATH 来操作构建中使用的库。

CMakeLists.txt文件修改后如下：

TODO 8: Click to show/hide answer
TODO 8: CMakeLists.txt
if (USE_MYMATH) add_subdirectory(MathFunctions) list(APPEND EXTRA_LIBS MathFunctions) list(APPEND EXTRA_INCLUDES "${PROJECT_SOURCE_DIR}/MathFunctions") endif()

现在，我们有了这两个列表，我们需要修改 target_link_libraries 和 target_include_libraries 来使用他们。

对于 target_link_libraries()，使用 EXTRA_LIBS 替代库的名字，如下所示：

TODO 9: Click to show/hide answer
TODO 9: CMakeLists.txt
target_link_libraries(Tutorial PUBLIC ${EXTRA_LIBS})

对于 target_include_directories() 和 EXTRA_INCLUDES 是一样的：

TODO 10: Click to show/hide answer
TODO 10: CMakeLists.txt
target_include_libraries(Tutorial PUBLIC "${PROJECT_BINARY_DIR}" ${EXTRA_INCLUDES})

注意：这些处理模块的经典方法。在步骤三中，将介绍现代处理方法。

对应的修改源码中的使用。首先在 tutorial.cxx中，通过 USE_MYMATH 宏来引入 MathFunctions.h头文件。

TODO 11: Click to show/hide answer
TODO 11: tutorial.cxx
#ifdef USE_MYMATH #include "MathFunctions.h" #endif

通过 USE_MYMATH 来定义使用哪个计算平方根的方法：

TODO 12: Click to show/hide answer
TODO 12: tutorial.cxx
#ifdef USE_MYMATH const double outputValue = mysqrt(inputValue); #else const double outputValue = sqrt(inputValue); #endif

在 TutorialConfig.h.in 中引入 USE_MYMATH，以便于源码引用。

TODO 13: Click to show/hide answer
TODO 13: TutorialConfig.h.in
#cmakedefine USE_MYMATH

以上修改都完成后，无论是否使用这个库都是可选的。
